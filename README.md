## Test a Node RESTful API with Mocha and Chai

## Start MongoDB:
```
$ mongod
```

## Run Server:
```
$ npm start
```

## Run Test:
```
$ npm test
```

## Registers the following routes:
```
GET /book
GET /book/:id
POST /book
PUT /book/:id
DELETE /book/:id
```

## Demo:
This site was built using [Heroku](https://node-api-tdd.herokuapp.com/).
